# 사용자 여정

이 진단 가이드는 사용자의 애플리케이션에서 문제를 진단할 때 지침을 제공하기 위해 [진단 워킹 그룹](https://github.com/nodejs/diagnostics)에서 만들었습니다.

이 문서 프로젝트는 사용자 여정을 기반으로 구성되었습니다. 이러한 여정은 사용자가 문제를 근본적으로 해결하기 위해 따를 수 있는 일관된 단계별 절차 집합입니다.
